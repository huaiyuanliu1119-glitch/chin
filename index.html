氣泡排序法

原理

氣泡排序法是一種簡單的排序演算法。它會反覆地遍歷要排序的資料列，依次比較相鄰的兩個元素，若順序錯誤則交換它們的位置。這個過程會重複進行，直到沒有需要交換的元素為止。

氣泡排序的名稱來源於較大的元素會逐漸「浮」到資料列的頂端，就像氣泡上升到水面一樣。

應用

氣泡排序法適用於小型資料集的排序，因為它的時間複雜度為 O(n²)，對於大型資料集效率較低。它的主要應用包括：

  • 學習排序演算法的基礎概念
  • 需要簡單實現的情境
  • 小型資料集的排序

範例程式碼

<script>
    // 氣泡排序函式
    function bubbleSort(arr) {
        let n = arr.length;
        for (let i = 0; i < n - 1; i++) {
            for (let j = 0; j < n - i - 1; j++) {
                if (arr[j] > arr[j + 1]) {
                    // 交換元素
                    let temp = arr[j];
                    arr[j] = arr[j + 1];
                    arr[j + 1] = temp;
                }
            }
        }
        return arr;
    }
</script>
